#### 开卡方式分别是什么意思？

​              ①指定天数内：会员购买会员卡后最迟开卡的期限，再次期限内入场即开卡，开卡日期                        

​                                         就从入场当天算起；若会员在此开卡期限内一直未到店开店，开卡期限                      

​                                         最后一天会员卡自动开卡；

​              ②买卡当天：买卡当天即开卡；

​              ③指定日期开卡：指定某一天开卡；

​              ④第一次入场：第一次入场即开卡；

#### 必须指定开卡方式吗？

不是的，可以不指定，选择--即可
