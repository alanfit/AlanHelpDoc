# 实用教程 | 如何给会员办卡？

///阿懒有非常强大、灵活的会员卡管理功能，支持期限卡、次卡和储值卡，支持的卡种如下：
日卡
月卡
周卡
月卡
年卡
家庭卡、情侣卡
储值卡
次卡
通过卡种的灵活设置，可以生成千变万化的购卡订单。卡种为用户购卡的最小单位，如，用户买年卡，那最小单位是一年卡，用户可以选择多个最小单位，如买两年卡，三年卡。月卡的最小单位是一个月，其他卡种以此类推。

///新增卡种
进入总部系统→卡种管理→新增

可选择次卡、日卡、月卡、年卡、附属卡
其中充值卡（储值卡），与其他几张卡不同的是，它可以用于消费，和商品销售搭配使用。储值卡的新增和使用在下面这篇文章中有详细介绍👇

●实用教程|教您如何使用阿懒系统储值卡模块

///给会员办卡
进入会员详情页→点击“办卡”
为会员选择需要的卡种类型，系统默认最多开卡时间是30天，如果超过30天需要在高级权限中设置。

若开卡的时候需要给会员优惠，例如：给会员办理一个30个月的月卡，送2个月。配置方法：把卡种类型选择为“月卡”，“时限”改为30，然后在“优惠方案”选择相应的优惠，这需要提前配置好收费方案。如何配置？请往下看↓↓↓

///新增收费方案
进入总部系统→运营管理→收费方案→新增

优惠方式可以选择优惠金额、赠送金额、赠送天数、赠送数量、打折、请假、手续费、预售定金。若是办30个月的月卡，赠送2个月，配置方法：“优惠方式”选择赠送天数然后把“赠送量”改为30天。

///购买附属卡

卡种类型选择“附属卡”，输入主卡的卡号。在阿懒系统中购买附属卡不需要付钱，附属卡的消费金额全部由主卡承担。

///最后
会籍卡相当于俱乐部的特殊门票，俱乐部给会员提供不同种类的会籍卡，使会员的选择更多，体验更好。另外，会籍卡搭配阿懒系统的盲刷特技，2秒入场，避免高峰期会员排队。
